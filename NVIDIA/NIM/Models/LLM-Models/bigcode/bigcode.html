Models#

  1. bigcode / starcoder2-7b

StarCoder2 Model Overview StarCoder2-7b is a state-of-the-art language model with 7 billion parameters, trained on 17 programming languages using The Stack v2 dataset. It employs advanced techniques like Grouped Query Attention and sliding window attention to enhance its performance on coding tasks. The model is optimized to handle a context window of 16,384 tokens and was trained using the Fill-in-the-Middle objective on 3.5+ trillion tokens.

StarCoder2-7b on Hugging Face StarCoder2-7b endpoint

Model Architecture:

Architecture Type: Transformer decoder

Supported Hardware Platform(s): NVIDIA L4 GPUs

Inference:**#

Engine: Triton Inference Server Test Hardware: NVIDIA L4 systems

Request#

import requests

url = "https://integrate.api.nvidia.com/v1/completions"

payload = {
    "model": "bigcode/starcoder2-7b",
    "prompt": "# Python function that finds primes:",
    "max_tokens": 200,
    "temperature": 0.1,
    "top_p": 1,
    "frequency_penalty": 0,
    "presence_penalty": 0,
    "seed": 0
}
headers = {
    "accept": "application/json",
    "content-type": "application/json",
    "authorization": "Bearer nvapi-xxxxxx"
}

response = requests.post(url, json=payload, headers=headers)

print(response.text)
